PM1 - Presentación de datos: Histograma, Polígono de frecuencias, Ojiva

Ya vimos la representación de datos (continuos y discretos) por medio de tablas de distribución de frecuencias, y aunque son utiles para el analísis de los datos, existen métodos gráficos para representar dichas tablas de frecuencias, lo cual nosfacilita el análisis de dichos datos.

Comencemos con el Histograma, el Polígono de frecuencias, y la Ojiva.


Histograma Un histograma es una gráfica de columnas que representa la distribución de frecuencias de datos continuos. Se utiliza para ver cómo se distribuyen estos últimos, así como qué clases tienen mayor concentración de datos.

Trazo de un histograma
Para trazar un histograma se siguen estos pasos: 

  • En una hoja de excel se elabora la tabla de distribución de frecuencias.
  • Se selecciona la columna de marcas de clase y la columna de frecuencia absoluta (o la de frecuencia relativa).
  • Se elige la petaña del menu Insertar, elegimos el bóton Insertar gráfico de columnas o de barras, seleccionamos la opcción Mas gráficos de columnas..., se elige el gráfico que muestre las clases en el eje x.
  • Se edita el gráfico para modificar las leyendas de título y de los ejes.


Polígono de frecuencias Un polígono de frecuencias es una gráfica de línea que sirve para representar la distribución de frecuencias de datos continuos. Igual que el histograma, se utiliza para ver la forma de la distribución de los datos, así como la ubicación de la mayor concentración de éstos. Como ya se explicó, un histograma y un polígono de frecuencias sirven para ver la forma de la distribución de los datos. 

Trazo de un polígono de frecuencias
Para graficar un polígono de frecuencias se siguen estos pasos:

  • En una hoja de excel se elabora la tabla de distribución de frecuencias.
  • Se selecciona la columna de marcas de clase y la columna de frecuencia absoluta (o la de frecuencia relativa).
  • Se elige la petaña del menu Insertar, elegimos el bóton Insertar gráfico de líneas o de areas, seleccionamos la opcción Mas gráficos de líneas..., se elige el gráfico que muestre las clases  en el eje x.
  • Se edita el gráfico para modificar las leyendas de título y de los ejes.


OjivaUna ojiva es una gráfica de línea donde se representa la frecuencia relativa acumulada. 

Trazo de una ojiva
Para graficar una ojiva de frecuencias se siguen estos pasos:

  • En una hoja de excel se elabora la tabla de distribución de frecuencias. calculando además la frecuencia acumulada.
  • Se selecciona la columna de marcas de clase y la columna de frecuencia acumulada.
  • Se elige la petaña del menu Insertar, elegimos el bóton Insertar gráfico de líneas o de areas, seleccionamos la opcción Mas gráficos de líneas..., se elige el gráfico que muestre las clases en el eje x.
  • Se edita el gráfico para modificar las leyendas de título y de los ejes.
