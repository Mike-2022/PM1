PM1 - Presentación de datos: Diagrama de barras, Diagrama circular

Diagrama de barras Un diagrama de barras se utiliza para representar una distribución de frecuencias de datos discretos o categóricos. En el eje x se indican los valores o las categorías y el eje y (la altura) de las barras corresponden a la frecuencia absoluta.

Trazo de un diagrama de barras
Para trazar un histograma de barras se siguen estos pasos: 

  • En una hoja de excel se elabora la tabla de distribución de frecuencias.
  • Se selecciona la columna de valores ó categorías o de marcas de clase y la columna de frecuencia absoluta (o la de frecuencia relativa).
  • Se elige la petaña del menu Insertar, elegimos el bóton Insertar gráfico columnas o de barras, seleccionamos la opcción Mas gráficos de columnas..., se elige el gráfico que muestre las categorías o valores en el eje x.
  • Se edita el gráfico para modificar las leyendas de título y de los ejes.

Diagrama circular o de pastelUn diagrama circular o de pastel, igual que el diagrama de barras, se usa para representar una distribución de frecuencias de datos discretos o categóricos y, como su nombre lo indica, hay que trazar un círculo, en el que luego hay que dibujar divisiones (rebanadas) que representan la frecuencia absoluta o la frecuencia relativa.

Trazo de un diagrama circular
Para trazar un diagrama circular se siguen estos pasos: 

  • En una hoja de excel se elabora la tabla de distribución de frecuencias.
  • Se selecciona la columna de valores ó categorías o de marcas de clase y la columna de frecuencia absoluta (o la de frecuencia relativa).
  • Se elige la petaña del menu Insertar, elegimos el bóton Insertar gráfico circular o de anillo, seleccionamos la opcción Mas gráficos circulares..., se elige el gráfico que muestre las categorías o valores en las rebanadas en forma proporcional.
  • Se edita el gráfico para modificar las leyendas de título y de los ejes.
